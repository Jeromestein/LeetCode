Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, and `false` otherwise.

**Example 1:**

**Input:** s = "anagram", t = "nagaram"

**Output:** true

**Example 2:**

**Input:** s = "rat", t = "car"

**Output:** false

**Constraints:**

- `1 <= s.length, t.length <= 5 * 104`
- `s` and `t` consist of lowercase English letters.

**Follow up:** What if the inputs contain Unicode characters? How would you adapt your solution to such a case?

---

[Submissions](https://leetcode.com/problems/valid-anagram/submissions/?source=vscode "https://leetcode.com/problems/valid-anagram/submissions/?source=vscode") | [Solution](https://leetcode.com/problems/valid-anagram/solutions/?source=vscode "https://leetcode.com/problems/valid-anagram/solutions/?source=vscode")

```python

class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        n,m=len(s),len(t)
        if m!=n:
            return False
            
        s_cnt=Counter(s)
        t_cnt=Counter(t)

        for char, freq in s_cnt.items():
            if freq!=t_cnt[char]:
                return False

        return True

```

O(n)
O(1)
